BookStore

Projeto pessoal para aprendizagem de novas tecnologias, e para melhorar minhas habilidades de programador. Nesse projeto, eu ententi que é um pouco mais dificil aprender algumas coisas sozinho, pois tive problemas com algumas coisas como subir um banco de dados no heroku, porque o desenvolvedor que deu a aula usou um H2(banco de dados em memoria) e o Heroku aceita o H2, e eu estava usando o MySQL. Depois de alguns dias eu descobri que podemos usar o PostgreSQL no Heroku para salvar os dados fornecido pelo usuario. Além disso, outro problema que eu tive nesse projeto foi publicar no GitPage, porque eu estava errando a nomenclatura na hora de subir o site. Tirando isso, me deparei com alguns bugs que não tive tanto problema em resolver.

Tecnoligas utilizada:Java, Spring Boot e Angular.
